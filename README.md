Java OOP Assignment

Files:
1. LoopsConditionsArrays.java
   - Demonstrates loops, conditions, arrays.

2. Person.java
   - Encapsulation using private fields with getters and setters.

3. Student.java
   - Inheritance from Person.
   - Polymorphism by overriding display().

4. Main.java
   - Creates objects and demonstrates OOP concepts.

OOP Concepts Used:
- Class & Object: Person and Student objects are created.
- Encapsulation: Private fields accessed through getters/setters.
- Inheritance: Student extends Person.
- Polymorphism: Student overrides display() method.
